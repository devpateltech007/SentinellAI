📋 User & Organization (Blue)
Role defines permissions; User inherits role. Project is created by a user and may include multiple frameworks via a join table.
✓ Compliance & Frameworks (Green)
Framework (SOC2, ISO, HIPAA) defines Controls. Each Control specifies Requirements and collects Evidence_Items.
📦 Evidence & Artifacts (Orange)
Append-only table. Evidence_Item links to control and requirement. Content hashed (SHA-256) for deduplication. Source connector field traces origin (AWS, GitHub, IaC, Slack, etc.).
📜 Audit History (Red)
Append-only, immutable table. Status_History records every status change with timestamp, rationale, and evidence IDs. No updates or deletes—audit trail only.
